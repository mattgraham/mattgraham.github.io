Creating Github Pages

I am excited to re-mention that we’ve released the GitHub Page Generator and that we’ve created some amazing templates to go along with it, but to add to it we’re open sourcing some of these themes. We want you to have the ability to make changes, additions and even submit pull requests to these themes to better them for yourself and the community.

Key features of each theme
  • SCSS/Compass & CSS3
  • Responsive design/development
  • Lightweight
  • Dynamic Navigation on Leap Day (Based on your H1 and H2 tags)
